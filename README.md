# Ascended Demon
***

Ascended Demon is a minecraft mod which allows
the Player to experience being a Demon
in the minecraft world. This is done through the
implementation of Demon Hearts and Rituals and other Demonic Items
and structures.

Dependencies: [Geckolib](https://www.curseforge.com/minecraft/mc-mods/geckolib), [Curios API](https://www.curseforge.com/minecraft/mc-mods/curios), [PlayerAnimator](https://www.curseforge.com/minecraft/mc-mods/playeranimator)

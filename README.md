# DescendedCore
***

DescendedCore is a core minecraft mod for the Descended Angel, Ascended Demon
and Primal Beast mods, containing various shared interfaces and features between the mods
as well as the fundemental Void-related aspects of the three mods.


Dependencies: [Geckolib](https://www.curseforge.com/minecraft/mc-mods/geckolib), [Curios API](https://www.curseforge.com/minecraft/mc-mods/curios), [PlayerAnimator](https://www.curseforge.com/minecraft/mc-mods/playeranimator)
